## Diseño Responsivo

TeoCRM adopta una estrategia Mobile First.

Principios:

- Toda nueva pantalla debe ser usable desde un teléfono.
- No se crearán versiones independientes para móvil y escritorio.
- Se utilizarán los breakpoints estándar de Tailwind CSS.
- Los componentes deberán ser reutilizables y adaptativos.
- El contenido secundario podrá ocultarse o simplificarse en dispositivos pequeños cuando mejore la experiencia de uso.
